# 图片复刻说明

## 使用方法

1. **准备图片文件**
   - 将你的设计图片文件命名为以下任一名称：
     - `design.png`
     - `design.jpg`
     - `design.jpeg`
     - `ui.png`
     - `ui.jpg`
   - 放在项目根目录下（与 `image-viewer.html` 同目录）

2. **打开查看**
   - 双击打开 `image-viewer.html`
   - 图片会自动加载并在iPhone 16框架中显示
   - 图片会自动缩放适配屏幕

## 图片要求

- 推荐尺寸：394×852px（iPhone 16设计稿尺寸）
- 格式：PNG、JPG、JPEG
- 图片会按比例缩放，保持原始比例
